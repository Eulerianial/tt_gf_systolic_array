# RTL Simulation using cocotb 

## To get both icarus (iverilog) and cocotb, use the oss-cad-suite binary and put this in the path

`
export PATH="/home/vijay/Downloads/oss-cad-suite/bin/:$PATH"
`

For the simulation here, the version of oss-cad-suite used is
oss-cad-suite-linux-x64-20260210

# To run the simulation:
- Go to `test` directory

```
    make clean
    make sim
```
